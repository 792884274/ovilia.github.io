How to learn JavaScript in a week

So, you want to learn JavaScript in a week?

Yes, that's possible. And you can even learn JavaScript in an hour if you call writing a Hello World program so.

My favourite programing language was C++ when I met with JavaScript for the first time, which was two years ago. Like most of new JavaScript programers, I thought it was easy to get used to JavaScript.

I used JavaScript but was thinking in C++. When I met with JavaScript problems, I would thought of how to solve it in C++ and then search for a similar solution in JavaScript. Thus, without the thorough understanding of JavaScript, I used it for the last two years. Then, in a project I did in my internship in Intel recently, I noticed the importance of having a complete understanding of JavaScript.

I took the different ways of implemening inheritance as different approaches to the same target. But it turned out that they are different patterns used in different situations. And without the thorough understanding of JavaScript, I won't even notice that there's a problem, let alone solving it by searching.

Time for a further step

I noticed the treasure of JavaScript when reading Professional JavaScript for Web Developers. Now, decided to read JavaScript: The Good Parts and write blogs of it.

As the author said in preface:

This is a book about the JavaScript programming language. It is intended for programmers who, by happenstance or curiosity, are venturing into JavaScript for the first time. It is also intended for programmers who have been working with JavaScript at a novice level and are now ready for a more sophisticated relationship with the language. JavaScript is a surprisingly powerful language. Its unconventionality presents some challenges, but being a small language, it is easily mastered.

My goal here is to help youto learn to think in JavaScript. I will show youthe components of the language and start you on the process of discovering the ways those components can be put together.
